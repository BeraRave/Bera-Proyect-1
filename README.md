Gestor de torneos para programación competitiva

Programa que crea un sistema de Ranking para torneos de programacion competitiva de forma manual/automatica

#no supe que poner en "caracteristicas"

Se uso la blibioteca "fstream" para la lectura y escritura de archivos de textos

#no supe que poner en "Como compilar"

Este programa se puede usar de forma manual o automatica;
Manual: proporcionar los datos por terminal
Automatica: buscara un archivo de texto llamado "participantes.txt" con el siguiente formato:
" nombre_de_participante puntuacion_de_participante "
por linea
